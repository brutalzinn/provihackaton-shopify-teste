# Shopify Teste HelloWorld

## Startup non docker

1. install ruby
2. install ngrok
3. gem install shopify-cli
4. shopify login
5. shopify node serve
6. ngrok http 8081

## Startup with docker

1. install ruby
2. install ngrok
3. gem install shopify-cli
4. shopify login
5. shopify node serve
6. docker build -t <teste-helloworld> .
7. ngrok http 8081

## Requirements

1. ngrok
2. shopify-cli
3. NodeJs
